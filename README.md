# CMandel
Reworked Mandelbrot explorer
- Minimalist GUI
- Realtime* rendering
- Updated filters and expanded fractal types
- Color schemes

# Controls
W/A/S/D - Up/Left/Down/Right
Mouse left - Zoom in
Mouse right - Zoom out
-/+ - Decrease/Increase iterations

[Only if system is fast enough to allow it*]
